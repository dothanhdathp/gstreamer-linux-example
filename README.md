# GStreamer Cho Linux

Đây là dự án thử gstreamer cho **Linux**. Cái này có nhiều _plug-in_ và các thư viện phụ thuộc có thể kiểm tra tốt hơn là so với **Android**.

- Hệ điều hành đang sử dụng chạy trên lõi `**x86-64**`

## Tải về một số thư viện

- gstreamer1.0-plugins-base
- gstreamer1.0-plugins-good
- gstreamer1.0-plugins-bad
- gstreamer1.0-plugins-ugly
- gstreamer1.0-libav

Lệnh

```text
sudo apt update && sudo apt install gstreamer1.0-plugins-base gstreamer1.0-plugins-good gstreamer1.0-plugins-bad gstreamer1.0-plugins-ugly gstreamer1.0-libav
```

> Tôi không biết nó là cái gì nhưng mà tốt nhất là sử dụng để thử nghiệm là chính. Một số thư viện nếu không có chắc là sẽ tải riêng thôi.
